# Registro de Evaluaciones V10

Aplicación web para el registro y gestión de evaluaciones escolares.

## Características

- **Gestión de Cursos**: Crea, elimina y organiza cursos mediante drag & drop
- **Evaluaciones Dinámicas**: Añade/elimina evaluaciones con configuración personalizada
- **Cálculo Automático**: Calcula notas y promedios ponderados en tiempo real
- **Reportes Inteligentes**: Genera reportes por evaluación o promedio semestral
- **Personalización**: Configura título, año escolar, colores y datos del docente
- **Persistencia**: Guarda todos los datos localmente en el navegador
- **Compartir**: Envía reportes por WhatsApp o Correo electrónico

## Instalación y Uso

1. Descarga los archivos: `index.html`, `style.css`, `script.js`
2. Ábre `index.html` en cualquier navegador moderno
3. ¡Listo! No requiere servidor ni instalación adicional
